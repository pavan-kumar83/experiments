What is MERN Stack?
MERN stack is a set of JavaScript-based technologies used to build full-stack web applications — from the database to the frontend — using a single language: JavaScript.
What is FULL Stack?
Full stack means developing both the front end (what the user sees and interacts with) and the back end (server, database, application logic) of a software application.
